=====================================
📢 Telegram 多频道推送配置指南
=====================================

✅ 修改已完成！现在支持同时推送到多个 Telegram 频道

┌─────────────────────────────────────┐
│  如何配置多个频道                    │
└─────────────────────────────────────┘

步骤1：打开 .env 文件

步骤2：修改 TELEGRAM_CHAT_ID 这一行

【单个频道】（原有方式）
TELEGRAM_CHAT_ID=-4954749769

【多个频道】（新功能）⭐
TELEGRAM_CHAT_ID=-4954749769,-1234567890,-9876543210

注意：用英文逗号 , 分隔多个 Chat ID

┌─────────────────────────────────────┐
│  完整配置示例                        │
└─────────────────────────────────────┘

OPENAI_API_KEY=sk-proj-xxxxx...
TELEGRAM_BOT_TOKEN=7951477893:AAHcxAae...
TELEGRAM_CHAT_ID=-4954749769,-1234567890,-9876543210
PH_TOKEN=rnaSocGWAb...

┌─────────────────────────────────────┐
│  测试配置                            │
└─────────────────────────────────────┘

保存 .env 文件后，运行：
py main.py

你会看到：
📱 正在发送消息到 3 个 Telegram 频道...
  ✅ 发送成功 → -4954749769
  ✅ 发送成功 → -1234567890
  ✅ 发送成功 → -9876543210
📊 发送完成：成功 3 个，失败 0 个

┌─────────────────────────────────────┐
│  如何获取新频道的 Chat ID            │
└─────────────────────────────────────┘

方法1：使用 @userinfobot
  1. 在 Telegram 搜索 @userinfobot
  2. 转发频道消息给它
  3. 获得 Chat ID

方法2：使用 @getidsbot
  1. 在 Telegram 搜索 @getidsbot
  2. 将它添加到频道
  3. 在频道发消息
  4. Bot 回复 Chat ID

┌─────────────────────────────────────┐
│  添加新频道步骤                      │
└─────────────────────────────────────┘

1. 获取新频道的 Chat ID（例如：-9999888777）
2. 打开 .env 文件
3. 在 TELEGRAM_CHAT_ID 后面加逗号和新 ID：
   
   修改前：
   TELEGRAM_CHAT_ID=-4954749769
   
   修改后：
   TELEGRAM_CHAT_ID=-4954749769,-9999888777

4. 保存文件
5. 下次运行机器人时自动生效！

┌─────────────────────────────────────┐
│  注意事项                            │
└─────────────────────────────────────┘

✓ 确保 Bot 已加入所有频道
✓ 确保 Bot 有发送消息权限
✓ Chat ID 是负数要加 - 号
✓ 用英文逗号分隔，不要用中文逗号
✓ 修改后无需重启，下次运行自动生效

┌─────────────────────────────────────┐
│  使用场景                            │
└─────────────────────────────────────┘

• 测试频道 + 正式频道
• 不同团队的频道
• 主频道 + 备份频道
